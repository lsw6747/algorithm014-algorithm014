## 2、Code Style

Java、Python、...

* Google code style
* Fackbook
* Airbnb

home -> 行头，end -> 行尾

ctrl + home -> 页头，ctrl + end -> 页尾 

ctrl + left/right -> 左右移动一个单词

ctrl + <-/delete -> 删除一整个单词

ctrl + shift + up/down -> 上下移动代码

ctrl + e -> 打开最近经常访问的文件

shift + home/end/up/down -> 从当前位置选中到行头/行尾/上一行相同位置/下一行相同位置

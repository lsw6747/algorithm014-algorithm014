## 6、刷题技巧

### 6.1、切题四件套

* Clarification（看题明确）
* Possible solutions（给出所有可能）
  * compare（time/space）
  * optimal（加强）
* Coding（多写）
* Test cases

### 6.2、五毒神掌（五遍刷题法）

#### 1、刷题第一遍

* 5分钟：读题 + 思考（理解题目意思，思考解法，最多延长至15分钟）
* 直接看解法：注意！多解法，比较解法优劣（理解学习和运用算法，而不是创造）
* 背诵、默写好的解法（很重要）

#### 2、刷题第二遍

* 马上自己写 -> LeetCode提交
* 多种解法比较、体会 -> 优化

#### 3、刷题第三遍

* 过了一天之后，再重复做题
* 不同解法的熟练程度 -> 专项练习

#### 4、刷题第四遍

* 过了一周：反复回来练习相同的题目
* 不同解法的熟练程度 -> 专项练习

#### 5、刷题第五遍

* 面试前一周恢复性训练
